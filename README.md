# Snake-game
Mon premier projet en python : un jeu snake simple codé avec PyGame


# Installation 
1. Installer Python
2. Ouvrire invite de commande
3. git clone https://github.com/adm971/Snake-game.git
4. cd Snake-game
5. pip install -r requirements.txt
6. python main.py

Vous pouvez maintenant jouer avec les touches directionnelles
